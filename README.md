# coffee_analysis
A data analysis exploring why coffee prices keep rising — connecting global warming, supply constraints, surging demand, and inflation through 51 years of historical data and forecasts to 2040.

## The Question
While shopping at Walmart I noticed my favorite 
coffee had gone up $1.20. That sparked a bigger 
question — how long before coffee is unaffordable?

## What This Project Does
- Pulls data from NASA, FAO, Yahoo Finance, and BLS
- Builds a master dataset of 51 years of coffee data
- Forecasts prices, temperatures, and yields to 2040
- Visualizes everything in a Tableau dashboard

## Tools Used
Python, Pandas, Scikit-learn, yfinance, Tableau Public

## Data Sources
NASA GISTEMP, UN FAO, Our World in Data, 
US Bureau of Labor Statistics, Yahoo Finance

## How to Run
1. Install requirements: pip install pandas scikit-learn 
   yfinance matplotlib openpyxl certifi
2. Run coffee_analysis.ipynb top to bottom
3. Open Tableau and connect to the output CSVs
